## 🤖 AI 스웜 아키텍처

### 1️⃣ 정찰 스웜 (Scout Swarm)
> "AI가 밤새도록 당신을 위해 입찰 기회를 찾습니다"

- **탐색 에이전트**: 나라장터 실시간 크롤링, 25개 자체조달기관 스캔, 해외 조달시장(UNGM, SAM.gov) 모니터링
- **매칭 에이전트**: 기업 DNA 분석 → 입찰공고 자동 매칭 → 낙찰 가능성 점수화 (0-100점)
- **알림 에이전트**: 맞춤 알림(카카오톡, SMS, 이메일), 긴급 마감 경보

**차별점**: 키워드 검색이 아닌 **첨부파일 내용까지 AI가 이해**하고 능동적 발굴

### 2️⃣ 두뇌 스웜 (Brain Swarm)
> "수천 개의 과거 입찰을 학습한 AI가 최적 전략을 설계합니다"

- **디지털 트윈 입찰 시뮬레이터**
  - 과거 3년 낙찰 데이터, 경쟁사 패턴, 평가위원 성향 분석
  - Monte Carlo Simulation으로 최적 가격 도출
  - "투찰가 8,723만원 제안 시 낙찰 확률 91.3%" 식의 구체적 전략

- **핵심 기술**
  - GraphRAG: 조달 규정 간 관계를 지식그래프로 구축
  - Monte Carlo Simulation: 수천 번의 가상 입찰
  - Few-shot Learning: 우수 제안서 사례 학습

### 3️⃣ 실행 스웜 (Action Swarm)
> "AI가 직접 서류를 작성하고, 검증하고, 제출합니다"

- **작성 에이전트**: RFP 자동 분석 → 기술제안서 초안 생성 → 최적가 산정
- **검증 에이전트**: 참가자격 체크, 결격사유 점검, 필수 첨부서류 확인
- **제출 에이전트**: Computer Use 기반 나라장터 자동 제출 (Human-in-the-Loop)

### 4️⃣ AI 음성 에이전트 (Voice Agent)
> "전화 한 통으로 모든 조달 업무를 처리합니다"

- 500ms 이하 응답 지연
- 30개국 다국어 지원
- 24시간 무중단 운영

## 📈 기대 효과

### 중소기업 측면
- 조달 전문인력 0명으로도 입찰 참여 → **연간 5천만원+ 인건비 절감**
- 제안서 작성 시간 **80% 단축** (2주 → 3시간)
- **낙찰률 20-30% 향상** (데이터 기반 최적 전략)
- 해외 조달시장 진출 지원 (다국어 AI)

### 조달청/국가 측면
- 중소기업 공공조달 참여율 **50% 증가**
- 부적격/무효 입찰 **90% 감소** → 행정비용 대폭 절감
- AI 기반 조달 혁신 **세계 최초 사례** 창출
- K-조달 표준 해외 수출 가능성
