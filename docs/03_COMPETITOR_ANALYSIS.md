| 항목 | 내용 |
|------|------|
| **URL** | https://www.g2bplus.kr |
| **운영사** | 민간 기업 |
| **데이터 규모** | **31,037,349건** (+29,355 daily) |
| **데이터 소스** | 나라장터 + 25개 자체 조달기관 |
| **기술** | BigData, AI 기술 적용 |

#### 주요 기능
- ✅ 맞춤설정 입찰공고
- ✅ 맞춤설정 사전규격
- ✅ 입찰공고 모니터링
- ✅ 입찰공고 AI추천
- ✅ 맞춤형 알림서비스
- ✅ 투찰가격 모의산출
- ✅ 다양한 맞춤형 검색 옵션
- ✅ 경쟁업체 정보 & 영업활동 파악
- ✅ 메모 공유 및 협업
- ✅ 조달요청~계약까지 전과정 추적

#### 비즈니스 모델
- 기본 무료 + 프리미엄 유료 구독 (추정)

#### 강점
- 오랜 운영 경험
- 방대한 데이터 축적
- 25개 자체 조달기관 연동
- 안정적인 서비스

#### 약점
- 전통적인 웹 인터페이스
- AI가 보조 역할 (핵심 아님)
- 제안서 자동 생성 미제공
- 음성 인터페이스 없음

## ⚔️ K-조달 AI 스웜 차별화 전략

### 기술 차별화

| 항목 | 지투비플러스 | 비드넷 | 나라장터 | **K-조달 AI 스웜** |
|------|------------|--------|---------|------------------|
| **핵심 기술** | BigData + AI | 검색 | 데이터베이스 | **AI Agent Swarm** |
| **AI 역할** | 보조 (추천) | 제한적 | 없음 | **핵심 (자율 수행)** |
| **인터페이스** | 웹 | 웹 | 웹 | **웹 + 음성** |
| **자동화 수준** | 알림 | 알림 | 없음 | **End-to-End** |
| **제안서** | 미제공 | 미제공 | 미제공 | **자동 생성** |
| **시뮬레이션** | 투찰가 모의 | 없음 | 없음 | **Monte Carlo 디지털트윈** |
| **법령 해석** | 없음 | 없음 | 없음 | **GraphRAG 실시간** |
| **자동 제출** | 없음 | 없음 | 수동 | **Computer Use** |

### 기능 차별화

#### 🔵 K-조달 AI 스웜 Only
1. **AI 에이전트 스웜 아키텍처**
   - 다중 에이전트 협업
   - 창발적 지능 발휘
   - 24시간 자율 운영

2. **음성 에이전트**
   - 전화 한 통으로 모든 업무
   - 500ms 응답 속도
   - 30개국 다국어

3. **제안서 자동 생성**
   - RFP 분석 → 3시간 내 초안
   - Few-shot Learning 적용
   - 기업 맞춤형

4. **디지털 트윈 시뮬레이터**
   - Monte Carlo 10,000회 시뮬레이션
   - 91%+ 낙찰 예측 정확도
   - 시나리오별 최적 전략

5. **GraphRAG 법령 해석**
   - 조달 규정 지식그래프
   - 실시간 해석/적용
   - 규정 간 관계 분석

6. **Computer Use 자동 제출**
   - 나라장터 자동 로그인
   - 입찰서 양식 자동 작성
   - Human-in-the-Loop 승인

## 📊 SWOT 분석

### Strengths (강점)
- ✅ 조달청 최우수상 → 정부 연결고리, 신뢰도
- ✅ 고품질 MVP (89/100) 이미 존재
- ✅ 작동하는 Claude API 챗봇 (목업 아님)
- ✅ 반응형 디자인, SEO 최적화
- ✅ MCP 아키텍처 기반
- ✅ 차별화된 AI 스웜 컨셉

### Weaknesses (약점)
- ⚠️ Mock 데이터만 존재 (실제 API 연동 필요)
- ⚠️ 사용자 인증/DB 없음
- ⚠️ 음성은 텍스트 기반 (실제 음성 I/O 아님)
- ⚠️ 1인 개발 리소스 한계

### Opportunities (기회)
- 🚀 200조원 공공조달 시장
- 🚀 중소기업 시장 진입 장벽 완화 수요
- 🚀 정부 AI 서비스 확대 정책
- 🚀 조달청 협력 가능성 (공모전 연결)
- 🚀 AI 에이전트 시장 급성장

### Threats (위협)
- ⚠️ 나라장터 API 접근 권한 제한 가능성
- ⚠️ 대형 SI 기업 시장 진입
- ⚠️ 공공데이터 활용 규제
- ⚠️ 지투비플러스 등 기존 사업자 대응
