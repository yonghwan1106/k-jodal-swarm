## 🗓️ 개발 로드맵

### Phase 1: MVP 고도화 (2025 Q4 - 2026 Q1)
**기간**: 1-3개월

#### 목표
- Mock 데이터 → 실제 API 연동
- 핵심 기능 안정화
- 베타 테스트 준비

#### 세부 작업
| 주차 | 작업 내용 | 산출물 |
|------|----------|--------|
| 1-2주 | 나라장터 API 연동 (6개 서비스) | API 클라이언트 |
| 3-4주 | 실시간 입찰공고 수집 파이프라인 | Scout Swarm v1 |
| 5-6주 | 데이터베이스 구축 (Supabase) | DB 스키마 |
| 7-8주 | 사용자 인증 시스템 | Auth 시스템 |
| 9-10주 | 베타 테스트 (10개 기업) | 피드백 리포트 |
| 11-12주 | 버그 수정 및 안정화 | MVP v1.0 |

#### 예상 비용
- 서버/인프라: 월 10-20만원 (Vercel Pro, Supabase)
- API 비용: 무료 (공공데이터)
- 도메인/SSL: 연 5만원

### Phase 2: B2B SaaS 런칭 (2026 Q2)
**기간**: 4-6개월

#### 목표
- 유료 구독 모델 출시
- 초기 고객 100곳 확보
- 매출 발생

#### 세부 작업
| 월 | 작업 내용 | 산출물 |
|-----|----------|--------|
| 4월 | 구독 시스템 개발 | 결제 연동 |
| 5월 | 기업별 대시보드 | Multi-tenant |
| 6월 | 알림 시스템 (이메일, 카카오톡) | Notification |
| 7월 | Brain Swarm 고도화 | 시뮬레이터 v2 |
| 8월 | 마케팅 / 영업 시작 | 고객 획득 |
| 9월 | 고객 피드백 반영 | MVP v2.0 |

#### 가격 정책 (안)
| 플랜 | 월 가격 | 대상 | 기능 |
|------|--------|------|------|
| **Basic** | 9.9만원 | 1인 사업자 | 공고 모니터링, 알림, 기본 분석 |
| **Pro** | 29.9만원 | 중소기업 | + 시뮬레이터, 제안서 초안, 경쟁사 분석 |
| **Enterprise** | 99만원~ | 중견기업 | + 전담 지원, API 연동, 맞춤 개발 |

#### 예상 매출 (보수적)
- 6개월 차: 30개 고객 × 평균 20만원 = **600만원/월**
- 12개월 차: 100개 고객 × 평균 25만원 = **2,500만원/월**

## 🎯 Go-to-Market 전략

### 타겟 고객 세그먼트

#### 1차 타겟: IT/SW 중소기업
- **규모**: 10-50명
- **특징**: 공공SI 진출 희망, 기술력 있음
- **Pain Point**: 입찰 전문인력 부재, 규정 복잡

#### 2차 타겟: 스타트업
- **규모**: 10명 이하
- **특징**: 정부 R&D 과제 수주 희망
- **Pain Point**: 제안서 작성 경험 없음

#### 3차 타겟: 제조업 중소기업
- **규모**: 50-200명
- **특징**: 물품 납품 경험 있음
- **Pain Point**: 효율화 니즈

### 마케팅 채널

| 채널 | 전략 | 예상 효과 |
|------|------|----------|
| **콘텐츠 마케팅** | 입찰 노하우 블로그, 유튜브 | 오가닉 유입 |
| **SEO** | "나라장터 입찰", "공공조달" 키워드 | 검색 노출 |
| **커뮤니티** | 중소기업 커뮤니티, 창업 커뮤니티 | 입소문 |
| **제휴** | 창업지원기관, 중기부 | B2G 연결 |
| **공모전** | 정부/공공기관 공모전 활용 | 브랜딩 |

### 영업 전략

1. **무료 체험**: 14일 무료 체험 제공
2. **성공 사례**: 낙찰 성공 사례 적극 홍보
3. **입소문**: 초기 고객 추천 프로그램
4. **파트너십**: 세무사, 회계사, 법무사 제휴
