> 작성일: 2025-12-29

---

## 📌 API 개요

### 데이터 소스
- **공공데이터포털**: https://www.data.go.kr
- **조달정보개방포털**: https://data.g2b.go.kr
- **API 제공기관**: 조달청 (조달데이터관리팀)
- **관리부서 연락처**: 042-724-7258, 7677

### 공통 사항
| 항목 | 내용 |
|------|------|
| API 유형 | REST |
| 데이터 포맷 | JSON + XML |
| 비용 | **무료** |
| 심의유형 | 자동승인 (개발/운영 모두) |
| 기본 트래픽 | 개발계정 10,000회/일 |
| 트래픽 확장 | 운영계정 + 활용사례 등록 시 증가 가능 |

## 🔑 API 연동 설정

### 환경변수 설정 (.env.local)

```env
# 나라장터 API Keys
G2B_API_KEY=발급받은_API_키

# API Endpoints
G2B_BASE_URL=http://apis.data.go.kr/1230000

# 개별 서비스 경로
G2B_STANDARD_SERVICE=/ao/PublicDataOpenStdService
G2B_BID_SERVICE=/ao/BidPublicInfoService
G2B_AWARD_SERVICE=/ao/SuccbidInfoService  
G2B_CONTRACT_SERVICE=/ao/CntrctInfoService
G2B_PROCESS_SERVICE=/ao/CntrctProcssIntgOpenService
G2B_PRODUCT_SERVICE=/ao/PrdctInfoService
```

### 기본 요청 구조

```
GET {BASE_URL}{SERVICE_PATH}/{OPERATION}
  ?serviceKey={API_KEY}
  &pageNo={페이지번호}
  &numOfRows={조회건수}
  &type=json
  &{추가파라미터}
```

## ⚠️ 주의사항

1. **트래픽 제한**: 개발계정 기본 10,000회/일, 초과 시 운영계정 신청
2. **활용사례 등록**: 운영계정 트래픽 증가를 위해 활용사례 등록 필요
3. **데이터 갱신**: 실시간이 아닌 일정 주기로 갱신됨
4. **인코딩**: 한글 파라미터는 URL 인코딩 필요
5. **에러 처리**: API 응답 코드 및 에러 메시지 처리 필수
